# pr command

`git-workflow-helper` エージェントを使用し、以下の方針でcommit・PRを行う。

## ブランチ作成

| 現在のブランチ | 作成 |
| --- | --- |
| デフォルトブランチ | 差分に合った名称でブランチを作成 |
| epicブランチ | 差分に合った名称でブランチを作成 |
| 作業ブランチ | 作成しない（現在のブランチで継続） |

## コミット先

| 現在のブランチ | コミット先 |
| --- | --- |
| デフォルトブランチ | 新規作成したブランチ |
| epicブランチ | 新規作成したブランチ |
| 作業ブランチ | 現在のブランチ |

## PR

以下の方針でPRを発行する。

| 現在のブランチ | PRの向き先 |
| --- | --- |
| デフォルトブランチ | デフォルトブランチ |
| epicブランチ | epicブランチ |
| 作業ブランチ | デフォルトブランチ |
